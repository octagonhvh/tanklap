# Tanklap

* **Készítő neve:** Gazdag Gergő
* **URL:** https://octagonhvh.github.io/tanklap/index.html
* **Téma:** A weboldal egy kitalált múzeum, a "Tank Múzeum" virtuális bemutatója. A látogatók a II. világháborútól a hidegháború koráig terjedő időszakból, **3 ország** különböző tankjairól kapnak információt.
* **Saját JS:** A főoldalon egy kitölthető személyiségteszt, a kód a `scripts/kviz.js` helyen található meg. Egy `form`-ban `radio input`-ok segítségével érzékeli a javascript a felhasználó választásást, amit egy tömbben tárol és a teszt végén 4 megadott tank közül kiírja neki azt, amelyikhez a legtöbb pontot érte el.
* **Fonttípusok:** BBH Sans Bogle, minden oldalon a `<h1>`. Fjalla One, a `<h2>` tagek. Arimo, minden más szöveg.
